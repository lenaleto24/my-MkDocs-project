1. Откройте Telegram.

2. Перейдите в чат, где находится нужное сообщение — личный диалог, группу или канал.

3. Найдите сообщение, которое нужно закрепить.

4. Откройте меню действий с сообщением:

   - на смартфоне: нажмите и удерживайте сообщение;

   - на компьютере: нажмите правой кнопкой мыши по сообщению.

5. Выберите пункт **Закрепить**.

6. При необходимости настройте параметры закрепления:

   - показывать ли уведомление участникам;
  
   - для групп — закрепить для всех или только для себя.

7. Подтвердите действие.

Сообщение будет отображаться в верхней части чата и останется доступным, пока его не открепят.

Для просмотра всех закреплённых сообщений нажмите на область закрепления в верхней части чата.
